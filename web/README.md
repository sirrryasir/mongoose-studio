# Mongoose Studio Website

This directory contains the landing page and documentation site for [Mongoose Studio](https://mongoose-studio.yaasir.dev) (Updated for v1.2).

Built with Next.js, Tailwind CSS, and Framer Motion.

## Structure

- `app/page.tsx`: The main landing page.
- `app/docs`: Documentation pages.
- `public`: Static assets (screenshots, logo).
